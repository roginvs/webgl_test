# University assignment

Just a small playground for webgl

## Demo

<https://roginvs.github.io/webgl_test/>

## Running locally

### 1. Install nodejs

https://nodejs.org/en/

### 2. Install packages

```sh
npm install
```

### 3. Run local server

```sh
npm start
```
